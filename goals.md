# What I Want From a Job

**Replace this entire file with your actual goals and preferences.**

The OpenAI analyzer reads this file to evaluate whether a posting
aligns with what you're looking for. Be specific.

Consider including:
- Role types you want (IC, lead, contract, full-time, part-time)
- Technologies you want to work with
- Industries or domains of interest
- Remote / hybrid / on-site preference
- Compensation expectations or minimums
- Company size / stage preferences
- Deal-breakers (e.g., no crypto, no defense, must be remote)
- What "good work" looks like to you
- Career direction you're heading
